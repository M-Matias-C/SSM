## 📞 SUPORTE RÁPIDO

### Se algo não funcionar

#### Backend travou?
```bash
# Reiniciar
cd backend && node src/server.js
```

#### Frontend não carrega?
```bash
# Limpar cache
rm -rf frontend/.vite && npm run dev
```

#### MongoDB não conecta?
```bash
# Verificar
mongod --version

# Iniciar
mongod
```

#### Erro CORS?
```bash
# Verificar .env
cat backend/.env | grep FRONTEND_URL
# Deve ser: FRONTEND_URL=http://localhost:3000
```
